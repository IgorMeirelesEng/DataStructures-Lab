## Meu objetivo com esse repositório

O **DataStructures Lab** foi desenvolvido com um propósito de:

- Sintetizar a teoria mantendo o rigor e a formalidade matemática.
- Implementar estruturas de dados do **zero** (sem bibliotecas).
- Analisar a complexidade assintótica (Big-O, Θ, Ω)
- Traduzir teoria matemática em código **eficiente**.
- Construir base sólida para entrevistas técnicas e pesquisas na área de **Ciência de Dados**
- Desenvolver pensamento algorítmico.
